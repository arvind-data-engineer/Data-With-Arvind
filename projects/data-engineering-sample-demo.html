Project Folder

What you will see in the sample project

  • sample_data/sales.csv — raw sales transactions with order, product, and pricing details
  • sample_data/products.csv — product master data and category mappings
  • ingest_and_transform.py — Python ETL script for ingestion, validation, transformation, and output generation
  • output/ — target folder for cleaned `fact_sales.csv` and `sales_summary_by_category.csv`
How to run

Execute the sample ETL workflow

Run the script from the sample folder to generate the cleaned outputs.

cd projects/data-engineering-sample
python ingest_and_transform.py

The script writes two files into `projects/data-engineering-sample/output/`:

  • fact_sales.csv — cleaned and enriched sales fact data
  • sales_summary_by_category.csv — aggregated revenue summary by category
